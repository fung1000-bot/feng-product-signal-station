# 枫的产品信号站

每天从公开数据源中筛选与枫相关的产品、内容和 AI 编程信号，给出一个最值得验证的方向、Top 3 信号、反向视角和当天不该追的方向。

这不是新闻聚合。只有满足以下条件的晨报才会发布：

- 至少一个具体 build idea 或决策建议。
- 说明为什么是现在，以及为什么与枫相关。
- 提供可点击的原始来源。
- 保留公开来源中的真实收入、价格、互动量和其他判断数据。
- 不包含 API Key、个人隐私、原始日志、服务器信息或私人 profile。
